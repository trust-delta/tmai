# tmai へのコントリビューション

コントリビューションに興味を持っていただきありがとうございます。本リポジトリは **project hub** であり、ソースコードは含まれません。変更内容に応じた sub-repo に issue / PR を提出してください。

> **English version**: [CONTRIBUTING.md](./CONTRIBUTING.md)

## コントリビュート先

| 変更領域 | Sub-repo |
|----------|----------|
| サーバーロジック / オーケストレーション / MCP / HTTP / SSE 実装 | [`tmai-core`](https://github.com/trust-delta/tmai-core) (private、collaborator 権限必要) |
| React WebUI | [`tmai-react`](https://github.com/trust-delta/tmai-react) |
| Ratatui TUI クライアント | [`tmai-ratatui`](https://github.com/trust-delta/tmai-ratatui) |
| ワイヤー契約 — REST エンドポイント / CoreEvent variants / error taxonomy | [`tmai-api-spec`](https://github.com/trust-delta/tmai-api-spec) |

各 sub-repo に `CONTRIBUTING.md` があり、セットアップ・ビルド・PR 規約が記載されています。

## この hub repo が受け付ける変更

- `README.md` / `README.ja.md` — ランディングページ、sub-repo インデックス
- `assets/` — スクリーンショット、デモメディア
- `LICENSE`、`CHANGELOG.md` (split 前の履歴で凍結)
- リリース配布ワークフロー (公開後)

## Issue

ここに提出された issue は triage 後に適切な sub-repo へ transfer されます。高速化のため、可能であれば sub-repo に直接提出してください。

## 言語

Issues / Discussions は英語・日本語どちらでも構いません。
