# ブランチグラフ

GitKrakenに着想を得た、レーンベースの可視化によるインタラクティブなGitコミットグラフ。

<!-- screenshot: branch-graph.png -->

## 概要

ブランチグラフは、登録済みプロジェクトの完全なGit履歴をビジュアルグラフとして表示します。サイドバーでプロジェクトを選択すると表示されます。

## 機能

### レーンベースのコミットグラフ

各ブランチは固有の色を持つ独自の垂直レーンを持ちます。グラフには以下が表示されます:

- **コミットドット** -- 各レーン上の円（ブランチ先端はより大きく表示）
- **マージインジケーター** -- マージコミット用の二重円
- **フォーク/マージ線** -- レーン間の直線接続線
- **ブランチヘッダー** -- 各レーン上部にブランチ名とコミット数

### ブランチ階層

ブランチは親子関係で整理されます:

- **デフォルトブランチ** (main/master) -- 常に最左レーンに表示
- **現在のブランチ** (HEAD) -- 目立つインジケーターでハイライト
- **Worktreeブランチ** -- worktreeバッジ付き
- **通常ブランチ** -- 新しい順にソート
- **非アクティブブランチ** -- 最近のコミットがないブランチは別セクションに表示

### 折りたたみ可能なコミット

レーン上の中間コミットを折りたたんで視覚的なノイズを軽減できます:

- ブランチヘッダーの折りたたみトグルをクリックして展開/折りたたみ
- 折りたたみインジケーター（破線付き省略記号）でコミットが非表示になっている箇所を表示
- 折りたたみインジケーターをクリックして非表示コミットを展開

### リモートトラッキング

リモートトラッキングが設定されているブランチの場合:

- **先行数** -- まだプッシュされていないコミット数
- **遅延数** -- まだプルされていないコミット数
- **プッシュ/プルインジケーター** -- 同期状態を示す視覚的マーカー

### 深度警告

ブランチのネストが深くなりすぎた場合（ブランチからブランチを多数作成した場合）、ブランチ構造のフラット化を促す警告が表示されます。

## GitHub連携

PRとCIステータスがブランチグラフ上に直接オーバーレイ表示されます:

- **PRバッジ** -- レビューステータス（承認済み、変更要求、ドラフト）付きでブランチ先端に表示
- **CIチェックインジケーター** -- GitHub Actionsからの成功/失敗/保留中ステータス
- **リンクされたissue** -- ブランチ名やPRタイトルから自動抽出（例: `fix/issue-42` は issue #42 にリンク）

詳細は[GitHub連携](./github-integration.md)を参照してください。

## コミット詳細

コミットドットをクリックすると詳細が表示されます:

- 完全なSHAハッシュ
- コミットの件名と本文
- ブランチ名
- マージコミットフラグ

再度クリックするかEscapeキーで閉じます。

## ページネーション

グラフはデフォルトで限られた数のコミットを読み込みます。「もっと読み込む」をクリックすると追加の履歴を取得できます。

## アクション

グラフ内でブランチを選択すると、右側にアクションパネルが開き、以下の操作が可能です:

- **Diffを表示** -- ベースブランチとの差分を表示
- **エージェントを起動** -- ブランチのworktreeでAIエージェントを起動
- **Worktreeを作成** -- このブランチから新しいworktreeを作成
- **ブランチ/Worktreeを削除** -- 確認ダイアログ付き
- **AIマージ** -- AIエージェントにマージ操作を委任
- **AI PR作成** -- AIエージェントにPR作成を委任
- **チェックアウト** -- このブランチに切り替え
- **CIチェック** -- GitHub Actionsチェック結果の展開可能なリスト

## APIエンドポイント

| メソッド | パス | 説明 |
|----------|------|------|
| GET | `/api/git/branches` | 親子関係付きブランチ一覧 |
| GET | `/api/git/graph` | コミットグラフレイアウトデータの取得 |
| GET | `/api/git/log` | ブランチのコミットログ取得 |
| POST | `/api/git/fetch` | リモートからフェッチ |
| GET | `/api/github/prs` | ブランチのPR情報取得 |
| GET | `/api/github/checks` | CIチェックステータス取得 |
| GET | `/api/github/issues` | リンクされたissue取得 |
