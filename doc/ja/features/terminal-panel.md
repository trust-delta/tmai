# ターミナルパネル

xterm.jsとWebSocket I/Oによるブラウザ上のフルインタラクティブターミナル。

## 概要

ターミナルパネルは、AIエージェントのPTYセッションにWebSocket経由で接続されたリアルタイムターミナルを提供します。完全なANSIカラーレンダリング、キーボード入力、非ASCIIテキスト用のIMEをサポートします。

<!-- screenshot: terminal-panel.png -->

## 機能

### xterm.jsレンダリング

- **WebGLレンダラー** -- ハードウェアアクセラレーションによるターミナルレンダリング
- **ANSIカラー** -- 256色およびトゥルーカラーの完全サポート
- **スクロールバック** -- 5000行のスクロールバック履歴
- **スクロールバックリプレイ** -- 接続時に過去の出力を再生

### WebSocket I/O

ターミナルは `/api/agents/{id}/terminal` のWebSocket経由でエージェントのPTYに接続します:

- **サーバー → クライアント**: バイナリフレーム（ANSIエスケープ付きの生のPTY出力）
- **クライアント → サーバー**: バイナリフレーム（生のキーボード入力）
- **制御メッセージ**: JSONテキストフレーム（例: ターミナルリサイズ）

### IMEサポート

日本語やその他の非ASCII入力用:

- **Ctrl+I** でIMEオーバーレイを切り替え
- ヘッダーの **あ** ボタンをクリック
- オーバーレイにテキストを入力し、送信してPTYにテキストを送信

### ターミナルリサイズ

ブラウザウィンドウのサイズ変更時にターミナルは自動的にリサイズイベントを送信します:

```json
{
  "type": "resize",
  "cols": 120,
  "rows": 40
}
```

## ヘッダー

ターミナルヘッダーには以下が表示されます:

- **セッションID** -- PTYセッション識別子の先頭8文字
- **IMEトグル** -- IME入力オーバーレイの有効/無効切り替えボタン

## パススルー入力

フルターミナルビューなしで直接ターミナル入力を行うには、パススルーAPIを使用します:

```
POST /api/agents/{id}/passthrough
```

個別の文字や特殊キー（`Enter`、`Tab`、`Escape` など）の送信をサポートします。

## APIエンドポイント

| メソッド | パス | 説明 |
|----------|------|------|
| ANY | `/api/agents/{id}/terminal` | WebSocketターミナル接続 |
| GET | `/api/agents/{id}/output` | エージェント出力テキストを取得 |
| POST | `/api/agents/{id}/passthrough` | 生のターミナル入力を送信 |
