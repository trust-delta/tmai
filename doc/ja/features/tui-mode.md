# TUIモード

tmaiにはtmuxパワーユーザー向けのratatuiベースのターミナルUIが含まれています。デフォルトモードはWebUIですが、TUIも完全にサポートされたインターフェースとして継続しています。ターミナルから離れたくないユーザーに最適です。

## 起動

```bash
# tmuxが必要 — tmuxペイン内で実行
tmai --tmux
```

> **Note**: v0.20よりデフォルトモードがWebUIに変更されました。TUIは `--tmux` フラグで利用可能です。

## TUIが向いている場面

- 既にtmuxで作業しており、ターミナルから出たくない
- ブラウザが使えないSSHセッション
- ブラウザのオーバーヘッドなしで軽量な監視がしたい
- マウスよりキーボード駆動のワークフローを好む

## レイアウト

```
┌─────────────────┬─────────────────────────────────┐
│ Sessions        │ Preview                         │
│                 │                                 │
│ [◈Hook] main:0  │ Do you want to make this edit?  │
│   Claude Code   │                                 │
│   ⠋ Processing  │ ❯ 1. Yes                        │
│                 │   2. Yes, allow all...          │
│ [IPC] main:0.1  │   3. No                         │
│   Claude Code   │                                 │
│   ✳ Idle        │                                 │
└─────────────────┴─────────────────────────────────┘
 j/k:Nav 1-9:Select i:Input →:Direct ?:Help q:Quit
```

- **左パネル**: 検出ソースアイコンとステータス付きのエージェント一覧
- **右パネル**: 選択中エージェントのターミナル出力ライブプレビュー（ANSIカラー対応）
- **ステータスバー**: キーボードショートカットのヒント

## モード

### ノーマルモード

エージェントのナビゲーションとクイックアクション:

| キー | 動作 |
|------|------|
| `j` / `k`、`↓` / `↑` | エージェント選択 |
| `y` | 承認 / Yes |
| `n` | No（UserQuestionのみ） |
| `1-9` | 番号で選択肢を選択 |
| `Space` | 複数選択トグル |
| `f` | ペインにフォーカス |
| `x` | ペインを終了（確認あり） |
| `?` | ヘルプ |
| `q` / `Esc` | 終了 |

### 入力モード (`i`)

テキストを入力し、Enterで選択中のエージェントに送信。

### パススルーモード (`p` / `→`)

全キーイベントがエージェントのtmuxペインに直接転送されます。エージェント内でのインタラクティブな編集やスクロールに便利。`Esc` で終了。

## ビューオプション

| キー | 動作 |
|------|------|
| `Tab` | ビューモード切り替え（Split / List / Preview） |
| `l` | 分割方向切り替え（横 / 縦） |
| `Ctrl+d` / `Ctrl+u` | プレビュースクロール |

## その他の機能

| キー | 動作 |
|------|------|
| `t` | タスクオーバーレイ（チームメンバー選択時） |
| `T` | チーム一覧画面 |
| `r` | モバイルリモート用QRコード表示 |
| `W` | エージェントをIPCラップで再起動 |
| `R` | Fresh Session Reviewを起動 |
| `U` | 使用量確認（Claude Max/Pro） |

## 設定

`~/.config/tmai/config.toml` のTUI固有オプション:

```toml
[ui]
show_preview = true        # プレビューパネルを表示
preview_height = 40        # プレビューの高さ（パーセンテージ）
color = true               # カラー表示を有効化
line_wrap = true           # プレビューで長い行を折り返す
```

すべてのコア機能（Hooks、Auto-approve、Teams、PTYラッピング）はTUIとWebUIの両方で同じように動作します。

## 今後の計画

TUIはWebUIと並行してメンテナンスを継続します。将来的にはWezTerm等の他のターミナルマルチプレクサにも対応し、tmux以外でもTUIを利用できるようになる予定です。
