# WebUI 概要

tmaiのデフォルトモードでは、AIエージェントの監視・操作用WebUIダッシュボードが起動します。

## アーキテクチャ

- **バックエンド**: Rust (axum) Webサーバー（REST API、SSE、WebSocketエンドポイント）
- **フロントエンド**: React 19 + TypeScript + Tailwind CSS
- **リアルタイム通信**: Server-Sent Events (SSE) によるエージェント/チーム/使用量のライブ更新
- **ターミナル**: WebGLレンダリング対応のxterm.js、WebSocket接続
- **認証**: トークンベース（Bearerヘッダーまたはクエリパラメータ）

## 起動方法

```bash
# WebUIを起動（デフォルトモード）
tmai

# Chrome/Chromiumがアプリモードで自動的に開きます:
# http://localhost:9876/?token=<generated-token>
```

ポートは `~/.config/tmai/config.toml` で設定可能です:

```toml
[web]
port = 9876
```

## レイアウト

<!-- screenshot: webui-layout.png -->

ダッシュボードの構成:

### サイドバー（左側）

- **ステータスバー** -- エージェント数、注意バッジ、設定/セキュリティボタン
- **エージェント一覧** -- プロジェクト別にグループ化された全検出エージェントとステータスインジケーター
- **ターミナル一覧** -- アクティブなターミナルセッション
- **使用量パネル** -- 下部に折りたたみ可能なトークン使用量メーター

### メインエリア（右側）

メインエリアはサイドバーでの選択に応じて変化します:

| 選択項目 | 表示内容 |
|----------|----------|
| エージェント | 承認コントロール、入力バー、ターミナル付きのエージェント詳細 |
| プロジェクト | GitHub連携とアクションパネル付きのブランチグラフ |
| Worktree | diffビューアと管理アクション付きのWorktree詳細 |
| Markdown | ファイルツリーとMarkdownプレビュー・エディター |

### 空の状態

何も選択されていない場合、メインエリアにはエージェントまたはプロジェクトの選択を促すウェルカムカードが表示されます。

## リアルタイム更新

tmaiは `GET /api/events` のServer-Sent Events (SSE) を使用して更新をプッシュします:

| イベント | 説明 |
|----------|------|
| `agents` | エージェントのステータス変更（処理中、アイドル、承認待ち） |
| `teams` | チームとタスクの更新 |
| `teammate_idle` | チームメンバーがアイドル状態に移行 |
| `task_completed` | チームタスクの完了 |
| `context_compacting` | エージェントのコンテキストウィンドウ圧縮 |
| `usage` | トークン使用量メーターの更新 |
| `worktree_created` | 新しいworktreeの作成 |
| `worktree_removed` | worktreeの削除 |
| `review_launched` | コードレビューの開始 |
| `review_completed` | コードレビューの完了 |

更新は即座にプッシュされます -- エージェントステータスのポーリングは不要です。

## エージェント操作

WebUIからは以下の操作が可能です:

- **承認** -- 承認ボタンをクリックするか `y` を送信
- **選択肢の選択** -- AskUserQuestionの番号付き選択肢をクリック
- **テキスト送信** -- 入力バーにテキストを入力してEnterキーを押下
- **ターミナル** -- xterm.jsによるフルインタラクティブターミナル
- **終了** -- エージェントを終了
- **エージェント間メッセージング** -- あるエージェントから別のエージェントにテキストを送信
- **自動承認** -- エージェントごとの自動承認オーバーライドを切り替え

## プロジェクト

プロジェクトディレクトリを登録すると、ブランチグラフとGitHub連携が有効になります:

1. ステータスバーの設定ボタン（⚙）をクリック
2. 設定パネルでプロジェクトパスを追加
3. プロジェクトがサイドバーに表示され選択可能に
