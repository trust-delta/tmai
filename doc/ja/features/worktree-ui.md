# Worktree管理

WebUIから直接Git worktreeを作成、管理、操作できます。

## 概要

Git worktreeを使用すると、複数のブランチを異なるディレクトリに同時にチェックアウトできます。tmaiはworktreeの管理とその中でのAIエージェント起動のためのビジュアルインターフェースを提供します。

## Worktreeの作成

1. サイドバーでプロジェクトを選択してブランチグラフを開く
2. ブランチをクリックして選択
3. アクションパネルで**Worktreeを作成**をクリック
4. 名前を入力（英数字、`-`、`_` のみ、最大64文字）
5. worktreeが `.claude/worktrees/<name>/` 以下に作成される

<!-- screenshot: worktree-create.png -->

または、最初にブランチを選択することで、特定のベースブランチからworktreeを作成することもできます。

## Worktreeパネル

ブランチグラフまたはサイドバーでworktreeをクリックすると詳細パネルが開きます:

<!-- screenshot: worktree-panel.png -->

### 表示情報

- **ブランチ名** -- worktreeの現在のブランチ
- **ダーティインジケーター** -- コミットされていない変更がある場合 `*` を表示
- **エージェントステータス** -- このworktreeでAIエージェントが実行中かどうか
- **リポジトリ名** -- 親リポジトリのパス
- **Diff統計** -- ベースブランチとの比較における挿入数、削除数、変更ファイル数

### アクション

| アクション | 説明 |
|------------|------|
| **エージェントを起動** | このworktreeでAIエージェント（Claude Code）を起動 |
| **削除** | worktreeを削除（ダーティなworktreeには強制オプションあり） |
| **Diffを更新** | ベースブランチとのdiffを再読み込み |
| **Diffを表示** | DiffViewerで完全なdiffを表示 |

## Git操作

アクションパネルからブランチ管理操作が可能です:

| 操作 | 説明 |
|------|------|
| **チェックアウト** | メインリポジトリをこのブランチに切り替え |
| **ブランチ作成** | 選択したブランチから新しいブランチを作成 |
| **ブランチ削除** | ブランチを削除（確認付き） |
| **フェッチ** | リモートからフェッチ |
| **プル** | 最新の変更をプル |
| **AIマージ** | コンテキスト付きでAIエージェントにマージを委任 |
| **AI PR作成** | AIエージェントにPR作成を委任 |

### AI委任操作

**AIマージ**と**AI PR作成**は、以下を含む事前構築されたプロンプトで新しいAIエージェントを起動します:

- ソースブランチとターゲットブランチの名前
- PR用の正しいベースブランチ
- コンフリクト解決の指示（マージの場合）

AIエージェントはコンフリクト解決を含め、操作を自律的に処理します。

## Diffビューア

worktreeとそのベースブランチ間の変更を表示します:

- ファイルレベルのdiff（挿入は緑、削除は赤で表示）
- 変更ファイル総数、挿入数、削除数のサマリー
- アクションパネルまたはWorktreeパネルの「Diffを表示」をクリック

## APIエンドポイント

| メソッド | パス | 説明 |
|----------|------|------|
| GET | `/api/worktrees` | 全worktree一覧 |
| POST | `/api/worktrees` | worktreeを作成 |
| POST | `/api/worktrees/delete` | worktreeを削除 |
| POST | `/api/worktrees/launch` | worktreeでエージェントを起動 |
| POST | `/api/worktrees/diff` | ベースブランチとのdiffを取得 |
| POST | `/api/git/branches/create` | 新しいブランチを作成 |
| POST | `/api/git/branches/delete` | ブランチを削除 |
| POST | `/api/git/checkout` | ブランチをチェックアウト |
| POST | `/api/git/fetch` | リモートからフェッチ |
| POST | `/api/git/pull` | リモートからプル |
| POST | `/api/git/merge` | ブランチをマージ |
