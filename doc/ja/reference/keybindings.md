# キーバインド一覧

## 基本操作

| キー | 動作 |
|------|------|
| `j` / `↓` | 次のエージェントへ |
| `k` / `↑` | 前のエージェントへ |
| `q` / `Esc` | 終了 |
| `h` / `?` | ヘルプ表示 |

## 承認操作

| キー | 動作 |
|------|------|
| `y` | 承認 / Yesを選択 |
| `n` | Noを選択（UserQuestionのみ） |
| `1-9` | 番号で選択肢を選択（ナビゲート＋確定） |
| `１-９` | 全角数字でも選択可能 |
| `Space` | 選択をトグル（複数選択モード） |
| `Enter` | 複数選択を確定 |

> **Note**: その他の選択は、入力モード(`i`)またはパススルーモード(`p`)を使用してください。

## 入力モード

| キー | 動作 |
|------|------|
| `i` | 入力モード |
| `p` / `→` | パススルーモード（直接キー入力） |
| `Esc` | モードを抜ける |

## ペイン操作

| キー | 動作 |
|------|------|
| `f` | ペインにフォーカス |
| `x` | ペインを終了（確認あり） |

## プレビュー

| キー | 動作 |
|------|------|
| `Ctrl+d` | 下スクロール |
| `Ctrl+u` | 上スクロール |

## ビュー・表示

| キー | 動作 |
|------|------|
| `Tab` | ビューモード切り替え（Split/List/Preview） |
| `l` | 分割方向切り替え（Horizontal/Vertical） |
| `s` | ソート切り替え（現在無効） |
| `m` | モニタースコープ切り替え（現在無効） |

## Agent Teams

| キー | 動作 |
|------|------|
| `t` | タスクオーバーレイを表示（チームメンバー選択時） |
| `T` | チーム一覧画面を表示 |

> **Note**: Agent Teamsは実験的な機能です。チーム統合が有効な間、ソートとモニタースコープは一時的に固定されています。

## Web Remote

| キー | 動作 |
|------|------|
| `r` | QRコード表示 |
