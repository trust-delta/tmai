# Keybindings

## Basic Operations

| Key | Action |
|-----|--------|
| `j` / `↓` | Next agent |
| `k` / `↑` | Previous agent |
| `q` / `Esc` | Quit |
| `h` / `?` | Show help |

## Approval Operations

| Key | Action |
|-----|--------|
| `y` | Approve (send Enter to confirm) |
| `1-9` | Select AskUserQuestion option |
| `１-９` | Full-width numbers also work |
| `Space` | Toggle for multi-select |

> **Note**: To reject or select other options, use number keys, input mode (`i`), or passthrough mode (`p`).

## Input Modes

| Key | Action |
|-----|--------|
| `i` | Enter input mode |
| `p` / `→` | Enter passthrough mode (direct key input) |
| `Esc` | Exit mode |

## Pane Operations

| Key | Action |
|-----|--------|
| `f` | Focus pane |
| `x` | Terminate pane (with confirmation) |

## Preview

| Key | Action |
|-----|--------|
| `Ctrl+d` | Scroll down |
| `Ctrl+u` | Scroll up |

## View / Display

| Key | Action |
|-----|--------|
| `Tab` | Toggle view mode (Split/List/Preview) |
| `l` | Toggle split direction (Horizontal/Vertical) |
| `s` | Toggle sort |
| `m` | Toggle monitor scope (All/Session/Window) |

## Web Remote

| Key | Action |
|-----|--------|
| `r` | Display QR code |
